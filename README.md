# vanilla-js-weather-app
Weather App is a high-performance weather dashboard engineered to demonstrate mastery of fundamental frontend technologies without the overhead of external frameworks. Built entirely with Vanilla JavaScript (ES6+), this application interfaces with the OpenWeatherMap API to deliver real-time meteorological data.
